Travail sur le cryptage de données

Travail détaillé efféctué sur Kali Linux sur les différents types de cryptage de données : Symétrique, Asymétrique. Voici une bref définition pour comprendre les différents types de chiffrements utilisés. Le cryptage symétrique utilise la même clé pour crypter et décrypter les données. En revanche, le cryptage asymétrique offre une sécurité plus solide en utilisant une paire de clés : une clé publique pour le cryptage et une clé privée pour le décryptage
